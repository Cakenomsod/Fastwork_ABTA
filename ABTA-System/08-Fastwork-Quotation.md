# ใบเสนอราคา Fastwork — ABTA Phase 1

> ไฟล์นี้จัดทำเพื่อคัดลอกไปกรอกในฟอร์มใบเสนอราคาแบบแบ่งชำระบน Fastwork  
> ช่องที่ยังไม่ทราบวันที่จริงจะเว้นว่างไว้ — กรอกเมื่อได้รับมัดจำงวดแรกแล้ว

**ราคารวมทั้งโครงการ:** 9,000 บาท  
**จำนวนงวด:** 3 งวด (30% / 30% / 40%)  
**ระยะเวลารวม:** ประมาณ 20–25 วัน นับจากได้รับมัดจำงวดแรก

---

## ส่วนที่ 1 — รายละเอียด

### 1. ชื่องาน

```
พัฒนาระบบบริหารจัดการสมาชิกและสัมมนา ABTA เชื่อมต่อ LINE OA (Phase 1)
```

---

### 2. รายละเอียดงาน

```
พัฒนาระบบบริหารจัดการสมาชิกของสมาคมการค้าผู้ประกอบการธุรกิจห้องเช่า (ABTA) เชื่อมต่อ LINE Official Account ครอบคลุม 6 ระบบหลัก ดังนี้

1) ระบบรับสมัครสมาชิก
- สมัครผ่านแบบฟอร์มออนไลน์ กรอกข้อมูลพร้อมแนบสลิปโอนเงินในขั้นตอนเดียว
- บันทึกข้อมูลสมาชิกลงฐานข้อมูล (ชื่อ, นามสกุล, เบอร์โทร, อีเมล, หน่วยงาน/ตึก, สถานะ, วันหมดอายุ)
- ออก Member ID ชั่วคราวทันทีหลังส่งฟอร์ม
- ออกใบเสร็จชั่วคราวหลังนายทะเบียนอนุมัติข้อมูล
- ดึงข้อมูลสมาชิกเดิมจากเบอร์โทรหรือ LINE User ID (ไม่ต้องกรอกซ้ำ)
- รองรับการต่ออายุสมาชิกผ่าน LINE OA

2) ระบบรับสมัครสัมมนา
- ลงทะเบียนเข้าร่วมสัมมนา
- ดึงข้อมูลสมาชิกเดิมอัตโนมัติ
- รองรับข้อมูลเพิ่มเติม เช่น ไซส์เสื้อ ประเภทอาหาร
- แสดงสถานะการสมัครและค่าลงทะเบียน

3) ระบบอัปโหลดหลักฐานการชำระเงิน (ตรวจสอบโดยเจ้าหน้าที่)
- สมาชิกอัปโหลดสลิปผ่านระบบหรือ LINE OA
- ระบบบันทึกและแจ้งเตือนเจ้าหน้าที่ทันที
- นายทะเบียนอนุมัติข้อมูลแล้ว → ออกใบเสร็จชั่วคราว
- เหรัญญิกตรวจสอบสลิปและเปลี่ยนสถานะการชำระเงิน
- ไม่รวมระบบ OCR/AI ตรวจสลิปอัตโนมัติ

4) ระบบหลังบ้าน (Back Office)
- จัดการข้อมูลสมาชิก ตรวจสอบเอกสาร อนุมัติสมาชิก
- นายทะเบียนอนุมัติข้อมูล + ออกใบเสร็จชั่วคราว / เหรัญญิกตรวจสลิป + ออกใบเสร็จตัวจริง
- เปลี่ยนสถานะสมาชิกและการชำระเงิน
- Dashboard สรุปข้อมูลเบื้องต้น
- รองรับบทบาท แอดมิน และ นายทะเบียน

5) ระบบแจ้งเตือนผ่าน LINE OA
- แจ้งสมาชิก: สมัครสำเร็จ, รับสลิปแล้ว, อนุมัติแล้ว, สถานะเปลี่ยน, เตือนก่อนหมดอายุ
- แจ้งเจ้าหน้าที่: สมาชิกใหม่, มีสลิปใหม่, รายการรออนุมัติ

6) ระบบตรวจสอบสถานะผ่าน LINE OA
- สมาชิกพิมพ์ "เช็คสถานะ" แล้วบอทตอบกลับอัตโนมัติ
- แสดง Member ID, สถานะสมาชิก, สถานะการชำระเงิน, สถานะสัมมนา, วันหมดอายุ, สถานะการต่ออายุ

เทคโนโลยีที่ใช้: React, Node.js, Firebase, LINE Messaging API

สิ่งที่ไม่รวมใน Phase 1 นี้:
- หน้าเว็บ Self-Service สำหรับสมาชิก (วางแผนใน Phase 2)
- Dashboard แยกสิทธิ์ 5 บทบาท (วางแผนใน Phase 2)
- สร้างเอกสาร PDF อัตโนมัติ / Export Excel (วางแผนใน Phase 2)
- QR Check-in หน้างาน, OCR ตรวจสลิป, Broadcast LINE (วางแผนใน Phase 3–4)
```

## หมายเหตุอื่น ๆ / สัญญา / เงื่อนไขข้อตกลง

```
1. ราคารวม 9,000 บาท ครอบคลุมเฉพาะ Phase 1 ตามรายละเอียดงานข้างต้น

2. ระยะเวลาดำเนินการ 20–25 วัน นับจากวันที่ได้รับมัดจำงวดแรก (2,700 บาท)

3. ลำดับการพัฒนา: ระบบเช็คสถานะผ่าน LINE OA จะพัฒนาก่อนเป็นอันดับแรกตามที่ลูกค้าร้องขอ

4. ลูกค้าเป็นผู้รับผิดชอบค่าใช้จ่ายบริการภายนอกหลังส่งมอบ ได้แก่:
   - LINE Official Account ประมาณ 400–1,500 บาท/เดือน (ตามแพ็กเกจ)
   - Firebase ฟรีถ้าไม่เกิน Free Tier หรือประมาณ 500+ บาท/เดือนหากใช้งานหนัก

5. ลูกค้าต้องเตรียมก่อนเริ่มงาน:
   - Gmail ใหม่ชื่อสมาคม (สำหรับสมัคร LINE OA)
   - ข้อมูลบัญชีธนาคารสำหรับรับโอน
   - ข้อมูลสมาชิกเดิม (ถ้ามี) สำหรับนำเข้าระบบ

6. ฟีเจอร์ที่ไม่รวมใน Phase 1 นี้ (สามารถพัฒนาเพิ่มใน Phase ถัดไป):
   - หน้าเว็บ Self-Service, Dashboard 5 บทบาท, PDF อัตโนมัติ, Export Excel
   - QR Check-in, OCR ตรวจสลิป, Broadcast LINE, Backup อัตโนมัติ

7. ผู้พัฒนายินดีให้บริการแก้ไขงานและดูแลระบบหลังส่งมอบ ตลอดการใช้งาน ภายในขอบเขตงานที่ตกลง

8. การสื่อสารระหว่างดำเนินงาน: Fastwork, LINE กลุ่มโครงการ, LINE ส่วนตัว
```
