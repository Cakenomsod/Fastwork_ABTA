# ภาพรวมโครงการ ABTA

## โครงการคืออะไร

พัฒนา **ระบบบริหารจัดการสมาชิก** ของสมาคมการค้าผู้ประกอบการธุรกิจห้องเช่า (ABTA) ครอบคลุม:

- ระบบสมัครสมาชิก
- ระบบลงทะเบียนสัมมนา
- ระบบหลังบ้าน (Back Office)
- เชื่อมต่อ **LINE Official Account (LINE OA)**

เพื่อให้สมาชิกดำเนินการและตรวจสอบสถานะได้ด้วยตนเอง ลดภาระงานของเลขานุการ นายทะเบียน และเจ้าหน้าที่

---

## เป้าหมายหลัก

1. **ลดภาระงาน** ของเลขานุการ นายทะเบียน และเจ้าหน้าที่
2. **ให้สมาชิกเช็คสถานะเองได้ตลอดเวลา** ผ่าน LINE OA
3. **ลดคำถามซ้ำ** เรื่อง:
   - สมัครสำเร็จหรือยัง
   - ชำระเงินแล้วหรือยัง
   - ได้เลขสมาชิกหรือยัง
   - ใบเสร็จออกหรือยัง
   - สมัครสัมมนาสำเร็จหรือยัง

### ตัวชี้วัดความสำเร็จ

ลดการตอบคำถามซ้ำของเจ้าหน้าที่ได้ **อย่างน้อย 70–80%**

---

## แนวคิดหลักของระบบ

```
สมัคร → ชำระเงิน → ได้สถานะสมาชิกชั่วคราว → ใช้สิทธิ์ได้ทันที
```

หลังสมัคร สมาชิกต้องได้รับภายใน **5 นาที**:

1. ข้อความยืนยันการสมัคร
2. Member ID ชั่วคราว
3. หลักฐานการรับชำระเงิน (เมื่อแนบสลิป)
4. สิทธิ์สมัครกิจกรรมหรือสัมมนาได้ทันที

## บริการหลังส่งมอบ

- แก้ไขงานได้จนกว่าลูกค้าพึงพอใจ (ภายในขอบเขตที่ตกลง)
- ให้คำปรึกษาการใช้งาน
