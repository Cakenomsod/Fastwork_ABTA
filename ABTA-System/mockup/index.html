📋 ภาพรวม Phase 1

LINE OA — สมาชิก: สมัคร, ยืนยันสมาชิกเก่า, เช็คสถานะ, ดูบัตร+ใบเสร็จ
Web / LIFF — ฟอร์มกรอกข้อมูล + อัปโหลดสลิป
Back Office — แอดมิน / นายทะเบียน / เหรัญญิก
6 ระบบใน Phase 1: (1) รับสมัครสมาชิก + ยืนยันสมาชิกเก่า (2) รับสมัครสัมมนา (3) อัปโหลดหลักฐานชำระเงิน (Manual) (4) Back Office (5) แจ้งเตือน LINE OA (6) เช็คสถานะ — ลำดับแรก
อัปเดต 13 ก.ค. 2569: Flow ผูก LINE สมาชิกเก่า · Exception (ส่งใหม่) · สิทธิ์สมาชิกชั่วคราวครบ · Edge cases ผ่านแชทแอดมิน
ไม่อยู่ใน Phase 1: OCR/AI, QR Check-in, Self-Service Web, PDF อัตโนมัติเต็มรูปแบบ, Export Excel

❌ นายทะเบียนไม่ผ่าน (ขั้นที่ 1)

ข้อมูลไม่ถูกต้อง → แจ้ง LINE OA · สมาชิกแก้ไขผ่าน LIFF · ออก Member ID ชั่วคราวใหม่ · ใบเสร็จยังไม่ออก · ไม่ส่งเหรัญญิก · วนกลับขั้นที่ 1

❌ เหรัญญิกไม่ผ่าน (ขั้นที่ 2)

สลิปไม่ถูกต้อง → ใบเสร็จ rejected · ออกเลขใบเสร็จใหม่ · คงเลขถาวร + สมาชิกสมบูรณ์ · ส่งสลิปใหม่ผ่าน LINE OA → วนกลับขั้นที่ 2

💬 Edge cases — แชทแอดมิน

ผูก LINE ผิดรายการ · legacy/LINE ซ้ำ · ยืนยันตัวตนไม่พบแต่เคยเป็นสมาชิก → ข้อความในระบบ + ติดต่อแอดมินผ่าน LINE กลุ่ม

🎫 สิทธิ์สมาชิกชั่วคราว

ใช้สิทธิ์ครบเหมือนสมาชิกสมบูรณ์ — ชั่วคราวหมายถึงรอตรวจสอบเลข/ใบเสร็จ ไม่ใช่จำกัดสิทธิ์ · สมัครสัมมนา member_free ได้ (ถ้ายังไม่หมดอายุ)
