# Phase 2–4 — แผนอนาคต (ยังไม่ยืนยัน)

> ฟีเจอร์ใน Phase นี้ **ยังไม่ได้ยืนยันงบหรือเริ่มพัฒนา**  
> ราคาเป็นการประเมินเบื้องต้น อาจปรับเมื่อใกล้ถึงช่วงนั้น

---

## Phase 2 — ฟีเจอร์เสริม

| รายการ | รายละเอียด |
|--------|-----------|
| **ราคา** | 5,000–7,000 บาท |
| **ระยะเวลา** | +10–15 วัน |

### ฟีเจอร์ที่เพิ่มจาก Phase 1

#### 1. หน้าเว็บ Self-Service

- สมาชิก Login เข้าหน้าเว็บ
- Dashboard ส่วนตัว แสดงสถานะครบทุกขั้นตอน
- ดูละเอียดกว่าเช็คผ่าน LINE OA
- ดาวน์โหลดเอกสาร

> Phase 1 ใช้เช็คสถานะผ่าน LINE OA แทน (รวมในราคาเดิม)

#### 2. Dashboard แยกสิทธิ์ (Multi-role)

> Phase 1 มีบทบาท แอดมิน + นายทะเบียน + เหรัญญิก แล้ว (แยกขั้นตรวจข้อมูล/การเงิน)  
> Phase 2 เพิ่ม Dashboard แยกสิทธิ์เต็มรูปแบบ:

| บทบาท | สิทธิ์หลัก |
|-------|-----------|
| เลขานุการ | จัดการข้อมูลสมาชิก, ส่งข่าวสาร, ดูรายงาน |
| นายทะเบียน | ตรวจสอบเอกสาร, อนุมัติ, ออกเลขถาวร |
| เหรัญญิก | ตรวจสอบการชำระเงิน, ออกใบรับเงิน/ใบเสร็จ |
| กรรมการ | ดูรายงานและสถิติ |
| ผู้ดูแลระบบ | จัดการระบบทั้งหมด |

#### 3. ระบบสร้างเอกสาร PDF อัตโนมัติ (template เต็มรูปแบบ)

> Phase 1 มี workflow ใบเสร็จชั่วคราว/ตัวจริง + แสดงใน LINE OA แล้ว  
> Phase 2 เพิ่มการ generate PDF อัตโนมัติ:

- ใบรับเงินชั่วคราว
- ใบเสร็จรับเงิน
- หนังสือรับรองสมาชิก
- หนังสือยืนยันการเข้าร่วมสัมมนา

#### 4. Export รายงานเป็น Excel

- Export ได้ทุกหน้ารายงาน
- รายได้ค่าสมาชิก / รายได้จากสัมมนา

### ฟีเจอร์ที่วางแผน

| ฟีเจอร์ | ราคาประเมิน | รายละเอียด |
|---------|------------|-----------|
| **QR Check-in หน้างาน** | 1,500–2,500 บาท | ออก QR ให้ผู้สมัคร, เจ้าหน้าที่สแกนเช็คอิน |
| **Member History** | 1,000–1,500 บาท | ประวัติสมาชิกย้อนหลัง |
| **Broadcast LINE OA** | 1,000–1,500 บาท | ส่งข่าวสาร/ประกาศผ่าน LINE จาก Back Office (กรอง Tag / ประเภทสมาชิก / สถานภาพ → เลือกสมาชิก → ส่ง) · 🔧 **กำลังทำล่วงหน้า** แต่**ยังนับเป็นงานเฟสนี้** (ไม่ย้ายเข้างบ Phase 1) · พอเริ่มเฟสนี้แจ้งลูกค้าว่าส่วนนี้ทำไปแล้ว — ดู [12-Scope-Update-2026-07-21.md](./12-Scope-Update-2026-07-21.md) |
| **Backup ข้อมูลอัตโนมัติ** | 500–1,000 บาท | สำรองข้อมูลเป็นระยะ |
| **OCR ตรวจสลิปอัตโนมัติ** | 2,000–4,500 บาท | Tesseract หรือ AI API |

### แพ็กเกจ OCR ที่เคยเสนอ (อ้างอิง)

| แพ็กเกจ | ราคา | วิธีทำงาน |
|---------|------|----------|
| B — AI Assist | 11,000 บาท | AI อ่านสลิป → แอดมินยืนยันครั้งสุดท้าย |
| C — Full Auto | 13,500 บาท | OCR อนุมัติอัตโนมัติทั้งหมด |

> ลูกค้าเลือกแพ็กเกจ A (แอดมินตรวจเอง) สำหรับ Phase 1
