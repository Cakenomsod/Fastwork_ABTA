## 6 ระบบใน Phase 1

### 1. ระบบรับสมัครสมาชิก

**Workflow ที่ตกลง (ปรับ 30 มิ.ย. 2569):**

```
กรอกข้อมูล + แนบสลิปโอนเงิน (ขั้นตอนเดียว)
    ↓
บันทึกลงฐานข้อมูล
    ↓
ออก Member ID ชั่วคราวทันที
    ↓
แจ้งยืนยันผ่าน LINE OA
    ↓
ส่งเข้าระบบหลังบ้าน → รอเจ้าหน้าที่ตรวจสอบ
```

**ข้อมูลที่เก็บ:**

- Member ID
- ชื่อ / นามสกุล
- เบอร์โทรศัพท์
- อีเมล
- หน่วยงาน / ตึก
- สถานะสมาชิก
- วันหมดอายุสมาชิก

**ฟีเจอร์เพิ่มเติม:**

- ดึงข้อมูลสมาชิกเดิมจากเบอร์โทรหรือ LINE User ID (ไม่ต้องกรอกซ้ำ)
- ต่ออายุสมาชิกผ่าน LINE OA
- แจ้งเตือนก่อนหมดอายุ 1 / 0.5 เดือน

---

### 2. ระบบรับสมัครสัมมนา

- ลงทะเบียนเข้าร่วมสัมมนา
- ดึงข้อมูลสมาชิกเดิมอัตโนมัติ
- รองรับข้อมูลเพิ่มเติม: ไซส์เสื้อ, ประเภทอาหาร, ข้อมูลพิเศษ
- แสดงสถานะการสมัคร
- แสดงค่าลงทะเบียน

---

### 3. ระบบอัปโหลดหลักฐานการชำระเงิน (Manual)

> ⚠️ Phase 1 **ไม่มี** OCR หรือ AI ตรวจสลิปอัตโนมัติ

**ฝั่งสมาชิก:**
- อัปโหลดสลิป (รวมในขั้นตอนสมัคร หรือส่งผ่าน LINE OA)
- ระบบบันทึกและแจ้งเตือนเจ้าหน้าที่

**ฝั่งเจ้าหน้าที่:**
- ตรวจสอบสลิป
- เปลี่ยนสถานะการชำระเงิน
- อนุมัติรายการ

---

### 4. ระบบหลังบ้าน (Back Office)

**บทบาทใน Phase 1:** แอดมิน + นายทะเบียน

| ฟังก์ชัน | รายละเอียด |
|---------|-----------|
| จัดการข้อมูลสมาชิก | CRUD ข้อมูลสมาชิก |
| ตรวจสอบเอกสาร | ดูสลิปและเอกสารแนบ |
| อนุมัติสมาชิก | อนุมัติสมาชิกใหม่ / ต่ออายุ / สัมมนา |
| เปลี่ยนสถานะ | อัปเดตสถานะสมาชิกและการชำระเงิน |
| Dashboard | สรุปข้อมูลเบื้องต้น |

> บทบาท 5 ตำแหน่ง (เลขานุการ, เหรัญญิก, กรรมการ ฯลฯ) → **Phase 2**

---

### 5. ระบบแจ้งเตือนผ่าน LINE OA

**แจ้งสมาชิก:**
- สมัครสมาชิกสำเร็จ
- ส่งหลักฐานการชำระเงินแล้ว
- ได้รับอนุมัติเป็นสมาชิก
- สถานะสมาชิกเปลี่ยน
- แจ้งเตือนก่อนหมดอายุ

**แจ้งเจ้าหน้าที่:**
- มีสมาชิกใหม่
- มีการส่งหลักฐานการชำระเงิน
- มีรายการรออนุมัติ

---

### 6. ระบบตรวจสอบสถานะผ่าน LINE OA ⭐ ลำดับแรก

> ลูกค้าขอให้พัฒนาส่วนนี้ **ก่อนเป็นอันดับแรก** (ประมาณ 5–6 วัน)

สมาชิกพิมพ์คำสั่ง เช่น **"เช็คสถานะ"** แล้วบอทตอบกลับอัตโนมัติ:

- Member ID
- สถานะสมาชิก
- สถานะการชำระเงิน
- สถานะการสมัครสัมมนา
- วันหมดอายุสมาชิก
- สถานะการต่ออายุสมาชิก

**ทางเลือกที่ตกลง:**
- ✅ แบบที่ 1: เช็คผ่าน LINE OA — **รวมใน Phase 1 ราคาเดิม**
- ❌ แบบที่ 2: หน้าเว็บ Self-Service — **ย้ายไป Phase 2**

## เป้าหมายเมื่อ Phase 1 เสร็จ

**สมาชิกสามารถ:**
- สมัครสมาชิก / สัมมนา
- ส่งหลักฐานการชำระเงิน
- ได้รับ Member ID ชั่วคราวทันที
- เช็คสถานะผ่าน LINE OA ตลอดเวลา
- ได้รับแจ้งเตือนก่อนหมดอายุและต่ออายุได้ทันเวลา

**เจ้าหน้าที่สามารถ:**
- ตรวจสอบข้อมูลสมาชิกและการชำระเงิน
- อนุมัติสมาชิกผ่านระบบหลังบ้าน
- ลดการตอบคำถามซ้ำเรื่องสถานะต่าง ๆ
