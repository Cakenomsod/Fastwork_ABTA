# Product

## Register

product

## Platform

web

## Users

**Primary — สมาชิกสมาคม ABTA (ผู้ประกอบการค้าอัญมณีและเครื่องประดับ):** ใช้งานผ่าน LINE OA / LIFF บนมือถือเป็นหลัก ต้องการสมัครสมาชิก ต่ออายุ เช็คสถานะ ดูบัตรสมาชิกและใบเสร็จอย่างรวดเร็วและน่าเชื่อถือ

**Secondary — เจ้าหน้าที่สมาคม (นายทะเบียน, เหรัญญิก, แอดมิน):** ใช้ Back Office บนเดสก์ท็อป/แท็บเล็ต ทำงานคิวตรวจข้อมูลสมาชิกและสลิป อนุมัติ/ปฏิเสธ จัดการเลขสมาชิกและใบเสร็จ

## Product Purpose

ระบบสมาชิก ABTA Phase 1 รองรับการสมัครสมาชิกใหม่ ยืนยันสมาชิกเก่า ผูก LINE UID ออกเลขสมาชิกชั่วคราว/ถาวร ตรวจสอบสลิปโดยเจ้าหน้าที่ และแสดงบัตรสมาชิก/ใบเสร็จให้สมาชิกผ่าน LINE

ความสำเร็จ = สมาชิกทำธุรกรรมได้ครบใน LINE โดยไม่สับสน เจ้าหน้าที่ตรวจคิวได้เร็วและลดความผิดพลาด เอกสารและสถานะสอดคล้องกันทุกช่องทาง

## Positioning

ระบบสมาชิกที่เชื่อถือได้สำหรับสมาคมการค้า — ชัดเจนเรื่องสถานะ เลขสมาชิก และใบเสร็จ ไม่ใช่แค่ฟอร์มออนไลน์

## Brand Personality

**น่าเชื่อถือ · เป็นระเบียบ · อบอุ่นแบบสมาคม**

เสียงภาษาไทยสุภาพ ตรงไปตรงมา เน้นความชัดเจนของสถานะและเลขที่ ไม่เล่นคำ ไม่กดดัน ให้ความรู้สึกองค์กรมืออาชีพที่ดูแลสมาชิก

## Anti-references

- Dashboard SaaS สีเทา-น้ำเงินทั่วไป (generic admin template)
- UI ที่ดู AI-generated: side-tab accent, colored glow บนพื้นมืด, gradient text, card grid ซ้ำๆ
- ฟอร์มยาวไม่มี progressive disclosure
- ข้อความสถานะคลุมเครือหรือใช้ศัพท์เทคนิค
- Modal/confirm แบบ `window.confirm` ไม่มี context

## Design Principles

1. **สถานะชัดก่อนสวย** — สมาชิกและเจ้าหน้าที่ต้องรู้ทันทีว่าอยู่ขั้นตอนไหน อนุมัติหรือรอ
2. **คิวเป็นศูนย์กลาง (Back Office)** — หน้าจอเจ้าหน้าที่ออกแบบรอบงานตรวจ ไม่ใช่ CRUD ทั่วไป
3. **ยืนยันก่อนทำลาย** — การลบ/เปลี่ยนเลข/อนุมัติสำคัญต้องมี confirm ที่ชัดเจน
4. **มือถือก่อน (ฝั่งสมาชิก)** — LIFF ใช้บน LINE บนมือถือเป็นหลัก touch target และ contrast ต้องผ่าน
5. **ความสอดคล้องของเอกสาร** — บัตรสมาชิก ใบเสร็จ และข้อความ LINE ใช้ visual identity เดียวกัน (เขียวสมาคม + ทอง)

## Accessibility & Inclusion

- เป้าหมาย **WCAG 2.1 AA** สำหรับข้อความและ interactive elements
- รองรับ `prefers-reduced-motion` สำหรับ animation
- ฟอร์มมี label ชัดเจน error message เป็นภาษาไทยที่ actionable
- Focus indicator มองเห็นได้สำหรับ keyboard navigation ใน Back Office
